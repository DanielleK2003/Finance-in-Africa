# Credit Risk Analysis

## Objective
Analyze loan data from Google Cloud Storage to understand borrower risk, default rates, and portfolio performance.
## Project Overview
This project simulates a **real-world credit risk analysis workflow** for financial institutions and fintech companies.  
It includes:
1. **Data cleaning and preparation**
2. **Portfolio risk analysis**
3. **Risk segmentation**
4. **Visualization of risk metrics**
5. **Credit risk prediction using a Logistic Regression model**
## Skills demonstrated
- SQL and data analysis
- Credit risk analytics
- Borrower segmentation
- Prediction
- Financial reporting
